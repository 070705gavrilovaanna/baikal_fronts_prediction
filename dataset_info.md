# Описание исходного датасета

## Структура
Столбцы: 2
1. date - дата в формате ГГГГ-ММ-ДД
2. temp - температура воздуха в градусах Цельсия

## Объём данных
Период: 1 января 2014 - 30 декабря 2025
Количество записей: 4,381 день
Пропуски: отсутствуют

## Особенности
- Ежедневные измерения без пропусков
- Данные отсортированы по возрастанию даты
- Полный непрерывный временной ряд

## Источник
Данные получены через NASA POWER API для координат озера Байкал (53.5, 108.0).
